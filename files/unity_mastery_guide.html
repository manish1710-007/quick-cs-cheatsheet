Component Architecture

Composition Over Inheritance
GameObject = Container
    └── Transform (always present)
    └── Mesh Renderer (visual)
    └── Collider (physics)
    └── Rigidbody (physics body)
    └── Custom Scripts (behavior)
    └── Audio Source (sound)

Component Pattern:
- Each component = single responsibility
- Communicate via GetComponent<T>()
- Use events/delegates for decoupling
- Avoid deep inheritance hierarchies

Basic Movement Patterns

// Simple movement
transform.position += direction * speed * Time.deltaTime;

// Translate (relative space)
transform.Translate(direction * speed * Time.deltaTime);
transform.Translate(Vector3.forward * speed, Space.World);

// Move towards target
transform.position = Vector3.MoveTowards(
    transform.position,
    target.position,
    speed * Time.deltaTime
);

// Smooth damp (like smooth follow)
private Vector3 velocity;
transform.position = Vector3.SmoothDamp(
    transform.position,
    target.position,
    ref velocity,
    smoothTime
);

// Lerp (linear interpolation)
transform.position = Vector3.Lerp(
    startPos,
    endPos,
    t  // 0-1 progress
);

Rotation Patterns

// Rotate around axis
transform.Rotate(0, rotationSpeed * Time.deltaTime, 0);

// Rotate around point
transform.RotateAround(
    centerPoint,
    Vector3.up,
    angle * Time.deltaTime
);

// Look at target
transform.LookAt(target);

// Smooth look at
Quaternion targetRotation = 
    Quaternion.LookRotation(target.position - transform.position);
transform.rotation = Quaternion.Slerp(
    transform.rotation,
    targetRotation,
    rotationSpeed * Time.deltaTime
);

// Rotate to face movement direction
if (movement != Vector3.zero) {
    transform.forward = movement;

DeltaTime Explained

Frame-Independent Movement
Frame Rate Independence Formula:
┌─────────────────────────────────────────────┐
│  movement = speed × Time.deltaTime          │
│                                             │
│  Why:                                       │
│  - Time.deltaTime = time since last frame   │
│  - Makes movement consistent across FPS     │
│                                             │
│  Example:                                   │
│  60 FPS: deltaTime = 0.0167s               │
│  30 FPS: deltaTime = 0.0333s               │
│                                             │
│  Both result in same units/second!          │
└─────────────────────────────────────────────┘

Time class properties:
  Time.deltaTime    - Frame time
  Time.fixedDeltaTime - Fixed physics step
  Time.time         - Time since game start
  Time.timeScale    - Scale time (pause/slow-mo)

Physics Movement (3D)

public class PlayerMovement : MonoBehaviour {
    [SerializeField] private float moveSpeed = 5f;
    private Rigidbody rb;
    
    void Awake() {
        rb = GetComponent<Rigidbody>();
    }
    
    void FixedUpdate() {
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");
        
        Vector3 movement = new Vector3(h, 0, v);
        
        // Velocity-based movement
        rb.linearVelocity = movement * moveSpeed;
        
        // Or add force
        rb.AddForce(movement * moveSpeed, ForceMode.VelocityChange);
    }
}

Physics Movement (2D)

public class PlayerMovement2D : MonoBehaviour {
    [SerializeField] private float moveSpeed = 5f;
    [SerializeField] private float jumpForce = 10f;
    private Rigidbody2D rb;
    
    void Awake() {
        rb = GetComponent<Rigidbody2D>();
    }
    
    void FixedUpdate() {
        float h = Input.GetAxis("Horizontal");
        
        // Preserve Y velocity for gravity
        rb.linearVelocity = new Vector2(
            h * moveSpeed, 
            rb.linearVelocity.y
        );
    }
    
    public void Jump() {
        rb.linearVelocity = new Vector2(rb.linearVelocity.x, 0);
        rb.AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
    }
}

Collision vs Trigger
┌─────────────────────────────────────────────────────────────────┐
│                    COLLIDER MODES                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  COLLISION (Collider)          TRIGGER (Is Trigger = true)     │
│  ─────────────────             ───────────────────────────     │
│  • Physical collision          • No physical interaction       │
│  • Objects block each other    • Objects pass through          │
│  • Uses OnCollision methods    • Uses OnTrigger methods        │
│                                                                 │
│  OnCollisionEnter()            OnTriggerEnter()                 │
│  OnCollisionStay()             OnTriggerStay()                  │
│  OnCollisionExit()             OnTriggerExit()                  │
│                                                                 │
│  USE CASES:                                                     │
│  Collision: Walls, floors, obstacles, pushable objects         │
│  Trigger: Pickups, checkpoints, area detection, cutscenes      │
│                                                                 │
│  REQUIREMENTS FOR COLLISION:                                    │
│  1. Both objects have colliders                                 │
│  2. At least one has Rigidbody                                  │
│  3. Neither is a trigger                                        │
│                                                                 │
│  REQUIREMENTS FOR TRIGGER:                                      │
│  1. At least one collider has "Is Trigger" = true              │
│  2. At least one has Rigidbody                                  │
└─────────────────────────────────────────────────────────────────┘

Input Buffer Pattern

public class InputBuffer : MonoBehaviour {
    private float bufferTime = 0.2f;
    private float jumpBufferCounter;
    
    void Update() {
        // Buffer jump input
        if (Input.GetButtonDown("Jump")) {
            jumpBufferCounter = bufferTime;
        } else {
            jumpBufferCounter -= Time.deltaTime;
        }
        
        // Execute if grounded and buffer active
        if (isGrounded && jumpBufferCounter > 0) {
            Jump();
            jumpBufferCounter = 0;
        }
    }
}

// Coyote Time (grace period after leaving ground)
private float coyoteTime = 0.1f;
private float coyoteCounter;

void Update() {
    if (isGrounded) {
        coyoteCounter = coyoteTime;
    } else {
        coyoteCounter -= Time.deltaTime;
    }
    
    if (Input.GetButtonDown("Jump") && coyoteCounter > 0) {
        Jump();
    }
}

Object Pooling

public class ObjectPool : MonoBehaviour {
    [SerializeField] private GameObject prefab;
    [SerializeField] private int poolSize = 10;
    
    private Queue<GameObject> pool = new Queue<GameObject>();
    
    void Awake() {
        for (int i = 0; i < poolSize; i++) {
            GameObject obj = Instantiate(prefab);
            obj.SetActive(false);
            pool.Enqueue(obj);
        }
    }
    
    public GameObject Get() {
        if (pool.Count == 0) {
            GameObject obj = Instantiate(prefab);
            pool.Enqueue(obj);
        }
        
        GameObject obj = pool.Dequeue();
        obj.SetActive(true);
        return obj;
    }
    
    public void Return(GameObject obj) {
        obj.SetActive(false);
        pool.Enqueue(obj);
    }
}

Coroutine Basics

using System.Collections;

public class CoroutineExample : MonoBehaviour {
    void Start() {
        StartCoroutine(MyCoroutine());
    }
    
    IEnumerator MyCoroutine() {
        Debug.Log("Start");
        yield return new WaitForSeconds(2f);
        Debug.Log("After 2 seconds");
        yield return null; // Wait one frame
        Debug.Log("Next frame");
    }
    
    // Stop coroutine
    void Stop() {
        StopAllCoroutines();
        // Or stop specific
        StopCoroutine(MyCoroutine());
    }
}

Yield Instructions

// Wait one frame
yield return null;

// Wait for seconds
yield return new WaitForSeconds(1.5f);

// Wait for seconds (unscaled time)
yield return new WaitForSecondsRealtime(1f);

// Wait until condition is true
yield return new WaitUntil(() => isReady);

// Wait while condition is true
yield return new WaitWhile(() => isPlaying);

// Wait for end of frame
yield return new WaitForEndOfFrame();

// Wait for fixed update
yield return new WaitForFixedUpdate();

// Start another coroutine
yield return StartCoroutine(OtherCoroutine());

// Async operation (scene loading)
yield return SceneManager.LoadSceneAsync("Level2");

Common Coroutine Patterns

// Fade effect
IEnumerator Fade(Image image, float duration) {
    float elapsed = 0f;
    Color start = image.color;
    Color end = new Color(start.r, start.g, start.b, 0f);
    
    while (elapsed < duration) {
        elapsed += Time.deltaTime;
        image.color = Color.Lerp(start, end, elapsed / duration);
        yield return null;
    }
}

// Spawn waves
IEnumerator SpawnWaves() {
    foreach (Wave wave in waves) {
        for (int i = 0; i < wave.count; i++) {
            SpawnEnemy(wave.enemyPrefab);
            yield return new WaitForSeconds(wave.spawnDelay);
        }
        yield return new WaitForSeconds(wave.waitAfter);
    }
}

// Countdown timer
IEnumerator Countdown(int seconds) {
    while (seconds > 0) {
        Debug.Log(seconds);
        yield return new WaitForSeconds(1f);
        seconds--;
    }
    Debug.Log("Done!");
}

// Extension method for coroutine to task
public static Task AsTask(this IEnumerator coroutine) {
    var tcs = new TaskCompletionSource<bool>();
    var runner = new GameObject("CoroutineRunner").AddComponent<CoroutineRunner>();

Audio Scripting

public class AudioManager : MonoBehaviour {
    [SerializeField] private AudioSource musicSource;
    [SerializeField] private AudioSource sfxSource;
    
    public void PlaySFX(AudioClip clip) {
        sfxSource.PlayOneShot(clip);
    }
    
    public void PlaySFX(AudioClip clip, float volume) {
        sfxSource.PlayOneShot(clip, volume);
    }
    
    public void PlayMusic(AudioClip clip, bool loop = true) {
        musicSource.clip = clip;
        musicSource.loop = loop;
        musicSource.Play();
    }
    
    public void StopMusic() {
        musicSource.Stop();
    }
    
    public void FadeMusic(float targetVolume, float duration) {
        StartCoroutine(FadeRoutine(targetVolume, duration));
    }
    
    private IEnumerator FadeRoutine(float target, float duration) {
        float start = musicSource.volume;
        float elapsed = 0f;
        while (elapsed < duration) {
            elapsed += Time.deltaTime;
            musicSource.volume = Mathf.Lerp(start, target, elapsed / duration);
            yield return null;
        }
    }
}

Sound Manager Pattern

public class SoundManager : MonoBehaviour {
    public static SoundManager Instance;
    
    [System.Serializable]
    public class Sound {
        public string name;
        public AudioClip clip;
        [Range(0f, 1f)] public float volume = 1f;
        [Range(0.1f, 3f)] public float pitch = 1f;
        public bool loop;
    }
    
    [SerializeField] private Sound[] sounds;
    private Dictionary<string, AudioSource> sources;
    
    void Awake() {
        Instance = this;
        sources = new Dictionary<string, AudioSource>();
        
        foreach (Sound s in sounds) {
            AudioSource source = gameObject.AddComponent<AudioSource>();
            source.clip = s.clip;
            source.volume = s.volume;
            source.pitch = s.pitch;
            source.loop = s.loop;
            sources[s.name] = source;
        }
    }
    
    public void Play(string name) {
        if (sources.ContainsKey(name))
            sources[name].Play();
    }
}

2D Camera Follow

public class CameraFollow2D : MonoBehaviour {
    [SerializeField] private Transform target;
    [SerializeField] private float smoothSpeed = 0.125f;
    [SerializeField] private Vector3 offset = new Vector3(0, 0, -10);
    [SerializeField] private bool useBounds;
    [SerializeField] private Vector2 minBounds, maxBounds;
    
    void LateUpdate() {
        if (target == null) return;
        
        Vector3 desiredPos = target.position + offset;
        Vector3 smoothedPos = Vector3.Lerp(
            transform.position,
            desiredPos,
            smoothSpeed
        );
        
        if (useBounds) {
            smoothedPos.x = Mathf.Clamp(smoothedPos.x, minBounds.x, maxBounds.x);
            smoothedPos.y = Mathf.Clamp(smoothedPos.y, minBounds.y, maxBounds.y);
        }
        
        transform.position = smoothedPos;
    }
}

public class ThirdPersonCamera : MonoBehaviour {
    [SerializeField] private Transform target;
    [SerializeField] private float distance = 5f;
    [SerializeField] private float sensitivity = 2f;
    [SerializeField] private float smoothTime = 0.1f;
    
    private float yaw, pitch;
    private Vector3 currentVelocity;
    
    void LateUpdate() {
        // Get mouse input
        yaw += Input.GetAxis("Mouse X") * sensitivity;
        pitch -= Input.GetAxis("Mouse Y") * sensitivity;
        pitch = Mathf.Clamp(pitch, -40f, 80f);
        
        // Calculate position
        Vector3 targetPos = target.position + Vector3.up * 2f;
        Vector3 offset = new Vector3(0, 0, -distance);
        Quaternion rotation = Quaternion.Euler(pitch, yaw, 0);
        
        Vector3 desiredPos = targetPos + rotation * offset;
        transform.position = Vector3.SmoothDamp(
            transform.position, desiredPos, ref currentVelocity, smoothTime
        );
        transform.LookAt(targetPos);
    }
}

public class FirstPersonController : MonoBehaviour {

    [SerializeField] private float mouseSensitivity = 2f;

    }
}

NavMesh AI

using UnityEngine.AI;

public class EnemyAI : MonoBehaviour {
    [SerializeField] private Transform target;
    [SerializeField] private float stoppingDistance = 2f;
    
    private NavMeshAgent agent;
    
    void Awake() {
        agent = GetComponent<NavMeshAgent>();
    }
    
    void Update() {
        if (target == null) return;
        
        float distance = Vector3.Distance(transform.position, target.position);
        
        if (distance > stoppingDistance) {
            agent.SetDestination(target.position);
            agent.isStopped = false;
        } else {
            agent.isStopped = true;
            // Attack or interact
        }
    }
    
    // Patrol pattern
    public Transform[] waypoints;
    private int currentWaypoint;
    
    void Patrol() {
        if (waypoints.Length == 0) return;
        
        agent.SetDestination(waypoints[currentWaypoint].position);
        
        if (!agent.pathPending && agent.remainingDistance < 0.5f) {
            currentWaypoint = (currentWaypoint + 1) % waypoints.Length;
        }
    }
}

3D Character Controller

public class CharacterMovement : MonoBehaviour {
    [SerializeField] private float speed = 6f;
    [SerializeField] private float jumpHeight = 2f;
    [SerializeField] private float gravity = -20f;
    [SerializeField] private Transform groundCheck;
    [SerializeField] private LayerMask groundMask;
    
    private CharacterController controller;
    private Vector3 velocity;
    private bool isGrounded;
    
    void Awake() {
        controller = GetComponent<CharacterController>();
    }
    
    void Update() {
        // Ground check
        isGrounded = Physics.CheckSphere(
            groundCheck.position, 0.4f, groundMask
        );
        
        if (isGrounded && velocity.y < 0)
            velocity.y = -2f; // Stick to ground
        
        // Movement
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");
        Vector3 move = transform.right * h + transform.forward * v;
        controller.Move(move * speed * Time.deltaTime);
        
        // Jump
        if (Input.GetButtonDown("Jump") && isGrounded)
            velocity.y = Mathf.Sqrt(jumpHeight * -2f * gravity);
        
        // Gravity
        velocity.y += gravity * Time.deltaTime;
        controller.Move(velocity * Time.deltaTime);
    }
}

📊 ScriptableObjects

What are ScriptableObjects?

Data Container Pattern
MonoBehaviour vs ScriptableObject:

MonoBehaviour:
├── Attached to GameObject
├── Exists in scene
├── Duplicated per instance
└── Has transform, lifecycle

ScriptableObject:
├── Asset file in Project
├── Exists independent of scene
├── Shared by multiple objects
└── Data-only, no transform

Benefits:
• Reduce memory usage
• Centralize configuration
• Easy to modify during play
• Create multiple variations

⚡ Performance Optimization

Common Performance Issues

Avoid in Update():
  • GetComponent<T>() calls - Cache in Awake
  • GameObject.Find() - Use references/tags
  • Instantiate/Destroy - Use object pooling
  • String concatenation - Use StringBuilder
  • LINQ in hot paths - Use for loops

Optimization Patterns

// BAD: GetComponent every frame
void Update() {
    GetComponent<Rigidbody>().AddForce(force);
}

// GOOD: Cache reference
private Rigidbody rb;
void Awake() => rb = GetComponent<Rigidbody>();
void FixedUpdate() => rb.AddForce(force);

// BAD: Find every frame
void Update() {
    Player p = GameObject.Find("Player").GetComponent<Player>();
}

// GOOD: Singleton or reference
public class Player : MonoBehaviour {
    public static Player Instance;
    void Awake() => Instance = this;
}

// GOOD: Object Pooling
public class BulletPool : MonoBehaviour {
    Queue<GameObject> pool = new Queue<GameObject>();
    public GameObject Get() => pool.Count > 0 ? 
        pool.Dequeue() : Instantiate(prefab);
    public void Return(GameObject obj) => pool.Enqueue(obj);
}
