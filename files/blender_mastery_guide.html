Transform Modifiers

Transform Precision Formula
G/R/S + Axis + Modifier = Precise Control

Axis Keys:
  X, Y, Z = Global axis
  X→X, Y→Y, Z→Z = Local axis
  Shift+X/Y/Z = Exclude axis

Modifiers:
  Shift = Fine control (10x slower)
  Ctrl = Snap to increment
  Alt = Transform origin
  Number = Exact value (e.g., G X 2.5)

Examples:
  G Z 5      → Move up exactly 5 units
  R X 90     → Rotate 90° on X axis
  S Shift+Z  → Scale X and Y (exclude Z)

Essential Mesh Modeling Workflow

1
Blockout - Start with basic primitives (Shift + A)
Use cube, cylinder, sphere to establish proportions
2
Refine Topology - Add detail with Loop Cuts (Ctrl + R)
Strategic edge loops define form and support subdivision
3
Shape - Extrude (E), Bevel (Ctrl + B), Inset (I)
Add complexity while maintaining quad-based topology
4
Clean Up - Merge vertices (M), Dissolve (Ctrl + X)
Remove doubles, fix n-gons, optimize geometry

Sculpting Workflow Tips

Professional Workflow:
  1. Start with base mesh or dynamic topology
  2. Block major forms with Clay Strips
  3. Refine shapes with Grab and Draw
  4. Add detail with DamStrip, Crease
  5. Smooth frequently (Shift)
  6. Use masking for isolation
⚠️ Performance Tip: For high-detail sculpting, enable Dyntopo (Dynamic Topology) for adding detail, or use Remesh (Voxel remeshing) to maintain even topology. Use the Remesh modifier for retopology-ready results.

🎯 PBR Material Formula: For realistic materials, always use Principled BSDF as your main shader. Connect Base Color, Metallic (0 or 1), Roughness (0-1), and Normal Map. Avoid Mix Shader for simple materials - Principled BSDF handles 95% of real-world surfaces.

Common Operations

import bpy
import bmesh

# Enter edit mode
bpy.ops.object.mode_set(mode='EDIT')

# Create new mesh
mesh = bpy.data.meshes.new("MyMesh")
obj = bpy.data.objects.new("MyObj", mesh)
bpy.context.collection.objects.link(obj)

# Using bmesh for geometry
bm = bmesh.new()
bmesh.ops.create_cube(bm, size=2)
bm.to_mesh(mesh)
bm.free()

# Apply transformation
bpy.ops.object.transform_apply(
    location=True,
    rotation=True,
    scale=True
)

🏆 Pro Tips & Workflow

Modeling Tips

  • Always model at real-world scale
  • Use mirror modifier for symmetrical objects
  • Add bevel modifier last for realistic edges
  • Check normals are facing outward (Shift + N)
  • Use edge split modifier for hard edges
  • Apply scale before sculpting (Ctrl + A)

Performance Tips

  • Use Simplify in viewport for complex scenes
  • Hide objects you're not editing
  • Use bounding box display for distant objects
  • Enable Persistent Data for faster animation renders
  • Use render regions for testing
  • Keep subdivision levels reasonable

Render Optimization

  • Use adaptive sampling
  • Enable denoising
  • Reduce bounces for indirect light
  • Use tile size that matches GPU (256-512)
  • Render in passes for compositing
  • Use light portals for interiors

Keyboard Mastery

  • Learn pie menus (Z, ~, ., ,)
  • Use Numpad for quick views
  • Master the G/R/S + axis workflow
  • Use Ctrl for snapping during transform
  • Remember Shift for precision
  • Customize shortcuts in Preferences
