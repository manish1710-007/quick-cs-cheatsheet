🎬 Macros - Automate Repetition

┌─────────────────────────────────────────────────────────────────────────┐ │ MACRO WORKFLOW │ ├─────────────────────────────────────────────────────────────────────────┤ │ │ │ 1. PLAN: Think through the actions needed │ │ │ │ 2. RECORD: qa ... q (record to register a) │ │ │ └── Stop recording │ │ └── Start recording │ │ │ │ 3. TEST: @a (execute once to test) │ │ │ │ 4. Executing Macros

CommandAction

Macro Examples

;; Convert CSV to formatted output
qa      " Start recording to a
0f,lct,: " Find first comma, change
f,lct,, Score: 
j0q     " Move to next line, stop
100@a   " Execute 100 times

Editing Macros

CommandAction
:put aPaste macro a to buffer
Edit the textModify as needed
"ayyYank line back to register a

Macros are just text in registers - edit them like any text!

Recursive Macros

;; Macro that runs until failure
qa              " Start recording
0dw             " Delete first word
j               " Move to next line
@a              " Recursive call
q               " Stop
@a              " Start execution

;; Runs until @a fails (no more lines)

Macro Best Practices

  • Use 0 to normalize position (start of line)
  • Use ^ for first non-blank
  • Use search /pattern instead of counts
  • Test with single execution first
  • Use n@: for command repetition
  • Save useful macros in vimrc

Useful Autocommands

;; Highlight on yank
augroup highlight_yank
    autocmd!
    autocmd TextYankPost * silent! lua
        vim.highlight.on_yank()
augroup END

;; Return to last edit position
augroup resume_position
    autocmd!
    autocmd BufReadPost *
        \ if line("'\"") > 1 && line("'\"") <= line("$")
        \ | exe "normal! g`\""
        \ | endif
augroup END

;; Auto-remove trailing whitespace
augroup trim_whitespace
    autocmd!
    autocmd BufWritePre * %s/\s\+$//e
augroup END

;; Filetype-specific settings
augroup file_types
    autocmd!
    autocmd FileType python setlocal ts=4 sw=4
    autocmd FileType javascript setlocal ts=2 sw=2
    autocmd FileType go setlocal ts=4 sw=4 noexpandtab
augroup END

Nvim-cmp (Autocomplete)

local cmp = require('cmp')

cmp.setup({
    mapping = cmp.mapping.preset.insert({
        ['<C-b>'] = cmp.mapping.scroll_docs(-4),
        ['<C-f>'] = cmp.mapping.scroll_docs(4),
        ['<C-Space>'] = cmp.mapping.complete(),
        ['<CR>'] = cmp.mapping.confirm({
            behavior = cmp.ConfirmBehavior.Replace,
            select = true
        }),
        ['<Tab>'] = cmp.mapping(function(fallback)
            if cmp.visible() then
                cmp.select_next_item()
            else
                fallback()
            end
        end, { 'i', 's' }),
    }),
    sources = cmp.config.sources({
        { name = 'nvim_lsp' },
        { name = 'buffer' },
        { name = 'path' },
    })
})

Variables

;; Variable Types
let g:global_var = 1     " Global
let l:local_var = 2      " Local to function
let s:script_var = 3     " Local to script
let v:vim_var = 4        " Vim predefined
let w:window_var = 5     " Window local
let t:tab_var = 6        " Tab local
let b:buffer_var = 7     " Buffer local

;; Environment
let $MYVAR = 'value'
echo $HOME

;; Register
let @a = 'text'         " Set register a
echo @a                 " Get register a

Functions

;; Define function
function! MyFunction(arg1, arg2)
    echo a:arg1 . ' ' . a:arg2
    return a:arg1 + a:arg2
endfunction

;; Call function
call MyFunction(1, 2)

;; With range
function! MyRange() range
    echo a:firstline . '-' . a:lastline
endfunction

;; Lambda (Vim 8+)
let Square = {x -> x * x}
echo Square(5)

;; Autoload function
" In autoload/myplugin.vim:
function myplugin#Init()
    " ...
endfunction

Conditionals & Loops

;; If-else
if condition
    " ...
elseif other
    " ...
else
    " ...
endif

;; For loop
for i in range(10)
    echo i
endfor

for item in items
    echo item
endfor

;; While loop
while i < 10
    let i += 1
endwhile

;; Try-catch
try
    " risky code
catch /Pattern/
    " handle error
finally
    " cleanup
endtry
